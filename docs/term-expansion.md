# Term expansion

This is the source of the enumeration files to be materialized by the [Term-Weaver](https://github.com/include-dcc/term-weaver) tool.

## [LinkML properties](https://linkml.io/linkml-model/latest/docs/ReachabilityQuery/) currently supported *
- source_ontology 
- source_nodes
- relationship_types 
    - only supporting rdfs:subClassOf
- is_direct
    - true/false
- include_self
    - true/false

* It does not currently support 'traverse_up'
